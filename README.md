# Forum Photo Post

通用图文论坛帖生成器 Skill。将照片转化为图文并茂的论坛帖子，支持数码开箱、美食打卡、旅行游记、车展报道等场景。

## 使用方法

在 WorkBuddy 中加载此 skill 后，只需提供照片目录和话题描述即可自动生成论坛图文帖。

## 目录结构

- `SKILL.md` — 主流程（分析→大纲→撰写→打磨→交付）
- `references/style-rules.md` — 少数派写作风格精简版
- `references/bbcode-rules.md` — BBCode 格式速查表
- `assets/article-template.html` — 响应式 HTML 文章骨架

## 适用场景

- 数码产品开箱评测
- 美食探店分享
- 旅行游记
- 车展/活动报道
- 摄影作品分享
